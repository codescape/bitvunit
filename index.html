--- layout: default title: Accessibility Testing Framework ---

BitvUnit is an open source accessibility testing framework that makes it easy to automate accessibility checking for pages and applications that run in a web browser.

The framework uses Apache Maven for build and dependency management. With the help of HtmlUnit the framework inspects the HTML pages under test. It provides adapters for Hamcrest matchers because of their expressive syntax.
