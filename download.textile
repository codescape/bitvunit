Please choose your preferred way to download BitvUnit.

h3. GitHub Download

Download the latest version of the framework from the GitHub download page:

* "bitvunit-core-0.5.0.jar":https://github.com/downloads/codescape/bitvunit/bitvunit-core-0.5.0.jar
* "bitvunit-core-0.5.0-sources.jar":https://github.com/downloads/codescape/bitvunit/bitvunit-core-0.5.0-sources.jar
* "bitvunit-core-0.5.0-javadoc.jar":https://github.com/downloads/codescape/bitvunit/bitvunit-core-0.5.0-javadoc.jar

You can find previous versions on the "GitHub download page":https://github.com/codescape/bitvunit/downloads.

h3. Maven Repository

BitvUnit is available from the "Maven Central Repository":http://repo1.maven.org/maven2/de/codescape/bitvunit/. To use BitvUnit in your Maven project put the following dependency into the <code><dependencies/></code> section of the pom.xml:

{% highlight xml %}
    <dependency>
      <groupId>de.codescape.bitvunit</groupId>
      <artifactId>bitvunit-core</artifactId>
      <version>0.5.0</version>
    </dependency>
{% endhighlight %}
