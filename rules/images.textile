Below you can find the documentation for all rules in the category Images.

h3. AlternativeTextForAreaOfImageMapRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.3</li>
    </ul>
</aside>

AlternativeTextForAreaOfImageMapRule ensures that every area of an image map created by the <code><area/></code> tag within the given HTML document provides an alternative text through its <code>alt</code> attribute.

h3. AlternativeTextForImageRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.1</li>
    </ul>
</aside>

AlternativeTextForImageRule ensures that every image created by the <code><img/></code> tag within the given HTML document provides an alternative text through it's <code>alt</code> attribute. This rule is already satisfied by an empty attribute <code>alt=""</code> since this is the encouraged way for images that are used for layout aspects.

h3. AlternativeTextForLinkedImageRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.6</li>
    </ul>
</aside>

AlternativeTextForLinkedImageRule ensures that every image created by the <code><img/></code> tag within the given HTML document that is part of a hyperlink created by the <code><a/></code> tag provides a non empty alternative text through it's <code>alt</code> attribute. <p/> A good example for a valid linked image should look like this: <pre><code><a href="contact.html"><img src="image.gif" alt="Contact me"/></a></code></pre>

h3. AvoidBorderAttributeForImageRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.8</li>
    </ul>
</aside>

AvoidBorderAttributeForImageRule ensures that no image on the current page under test uses the <code>border</code> attribute to assign a border to itself which is deprecated since HTML 4.01 and should be done through cascading style sheets.

h3. AvoidServerSideImageMapRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.4</li>
    </ul>
</aside>

AvoidServerSideImageMapRule ensures that there are no server side image maps created by the <code><img ismap="ismap"/></code> tag within the given HTML document because they are inaccessible by users who cannot use a mouse because of some disability.
