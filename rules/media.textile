Below you can find the documentation for all rules in the category Media.

h3. AlternativeTextForAppletRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.3</li>
    </ul>
</aside>

AlternativeTextForAppletRule ensures that every <code><applet/></code> within the given HTML document provides
 an alternative textual description to explain the <code><applet/></code> to everyone who is not able to view
 and display the <code><applet/></code> itself.

h3. AlternativeTextForObjectRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.4</li>
    </ul>
</aside>

AlternativeTextForObjectRule ensures that every <code><object/></code> within the given HTML document provides
 an alternative textual description to explain the <code><object/></code> to everyone who is not able to view
 and display the <code><object/></code> itself.
