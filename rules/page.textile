Below you can find the documentation for all rules in the category Page.

h3. LanguageForHtmlTagRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.2</li>
    </ul>
</aside>

LanguageForHtmlTagRule ensures that the given HTML document provides its language information through the <code>lang</code> attribute on the <code><html/></code> tag. <p/> A valid example of a document in German language would look like this: <pre><code><html lang="de">...</html&gt</code></pre>

h3. TitleForPageRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.6</li>
    </ul>
</aside>

TitleForPageRule ensures that the document has a non empty title defined by the <code><title/></code> tag to describe that page. This rule is satisfied by a <code><title/></code> tag that contains minimum one readable character inside it's body. <p/> A good example would look like this: <pre><code><title>BitvUnit - User Manual</title></code></pre>
