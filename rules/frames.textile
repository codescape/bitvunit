Below you can find the documentation for all rules in the category Frames.

h3. AlternativeTextForInlineFrameRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.6</li>
    </ul>
</aside>

AlternativeTextForInlineFrameRule ensures that every inline frame within the given HTML document provides an alternative textual description to explain the inline frame or a hyperlink to an alternative version of that inline frame.

h3. TitleForFrameRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.3</li>
    </ul>
</aside>

TitleForFrameRule ensures that every frame that is marked through the <code><frame /></code> element within the given HTML document provides a description of that frame through its <code>title</code> attribute.

h3. TitleForInlineFrameRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author: Stefan Glase</li>
        <li class="version">Since: 0.3</li>
    </ul>
</aside>

TitleForInlineFrameRule ensures that every inline frame that is marked through the <code><iframe /></code> element within the given HTML document provides a description of that inline frame through its <code>title</code> attribute.
