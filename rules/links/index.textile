h3. AnchorsWithoutTargetAttributeRule

<aside class="rule-meta">
    <ul>
        <li class="author">Author:  Stefan Glase</li>
        <li class="version">since Version:  0.5</li>
    </ul>
</aside>

 AnchorsWithoutTargetAttributeRule ensures that no anchor <code><a /></code> tag within the given HTML document contains the <code>target</code> attribute that forces a link to open in a specific or new window.
