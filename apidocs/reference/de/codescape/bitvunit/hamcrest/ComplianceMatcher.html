Class Overview

Hamcrest matcher to be used to run accessibility checks against a single Rule or a RuleSet.

Usage examples: 


 assertThat(supportedType, is(compliantTo(ruleSet)));
 assertThat(supportedType, is(compliantTo(rule)));

Have a look at HtmlPageUtil methods for supported types.
